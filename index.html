Introduction to HTML:

  • HTML stands for HyperText Markup Language.
  • The latest version of HTML is HTML5.
  • HTML is rendered by Web browsers.
  • HTML file contains .html extension.
  • HTML is a markup language which means it makes use of tags.

  • Types of HTML tags:

    1. Paired & Unpaired tags:

      • Paired tags [Have an opening and closing tag] (ex: <html></html>)
      • Unpaired tags [Tags which need not be closed] (ex: <br>)

    2. Self-Closing tags:


      Tags which open and close on their own.(ex: <img>)

    3. Utility-based tags:

      • Formatting tags [For the purpose of formatting text] (ex:<b></b>)
      • Structure tags [Help in maintaining structure of HTML document] (ex:<body></body>)
      • Control tags [Tags requiring user interaction to control its functionality] (ex: <a></a>)
